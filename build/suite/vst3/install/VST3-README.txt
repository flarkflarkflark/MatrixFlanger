# VST3 Plugin for flark's MatrixFilter
# Steinberg VST3 format plugin
# Compatible with all major DAWs

Plugin Information:
- Name: flark's MatrixFilter
- Vendor: flark
- Version: 1.0.0
- Format: VST3
- Category: Audio Effect

Features:
- Professional audio filtering
- Real-time matrix visualization  
- Spectrum analyzer
- 7 filter types
- Cross-platform compatibility

Installation:
- macOS: ~/Library/Audio/Plug-Ins/VST3/
- Windows: C:\Program Files\Common Files\VST3\
- Linux: ~/.vst3/ or /usr/lib/vst3/

For DAW compatibility and support:
https://flark.dev/matrixfilter
